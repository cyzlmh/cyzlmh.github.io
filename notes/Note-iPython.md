# Run command

```
!<command>
```
Change directory
```
%cd
```
Reset
```
%reset
```

# Run python file

```
%run <filename>
%paste	# run code in clipboard
```

# Variable information

```
<variable>?
```

display variables
```
%who, %whos 
```

previous output
```
_(one underscore)
__(two underscore)
```

# Logging

```
%logstart, %logstop, %logon, %logoff
```

Print input history
```
%hist
```

# Bookmark

Add bookmark
```
%bookmark <name> [dir name]
```

List all bookmarks
```
%bookmark -l
```
